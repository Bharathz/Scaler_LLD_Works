Lists in HTML

Unordered List

Items in an unordered list are displayed with bullet points.

types : Default - circle filled | square - square filled | disc - circle filled

  • Item 1
  • Item 2
  • Item 3

Ordered List

Items in an ordered list are displayed with numbers.

types : 1 - numbers | A - uppercase letters | a - lowercase letters | I - uppercase roman numerals | i - lowercase roman numerals

  1. First item
  2. Second item
  3. Third item

Description List

A description list is used to define terms and their descriptions.

Term 1
Description 1 of term 1
Description 2 of term 1
Term 2
Description of term 2
Term 3
Description of term 3

Line Break

This is the first line.
This is the second line after a line break
