Style tag should be added to the header



Type of css

  • Inline CSS - using style attribute inside html tags
  • Internal CSS - using style tag inside head element
  • External CSS - using external .css file linked to html using link tag

CSS Syntax

selector { property: value; }

Selector priority

  1. Inline CSS
  2. Internal CSS
  3. External CSS

decendent selector

A decendent selector selects all elements that are descendants of a specified element.

Child Selector

A child selector selects all elements that are direct children of a specified element.
