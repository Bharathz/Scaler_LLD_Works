CSS Selectors

CSS selectors are used to "find" (or select) the HTML elements you want to style.

Types of CSS Selectors

  • Universal Selector (*): Selects all elements on the page. 
    * {
    color: green;   
}
  • Type Selector: Selects all elements of a given type. 
    p {
    font-size: 18px;
}
  • Class Selector: Selects all elements with a specific class attribute. 
    .example { 
    background-color: yellow;
}
  • ID Selector: Selects a single element with a specific ID attribute. 
    #unique {
    border: 2px solid black;
}
  • Attribute Selector: Selects elements based on an attribute or attribute value. 
    a[target="_blank"] {   
    color: red;
}
  • Pseudo-class Selector: Selects elements based on their state. 
    a:hover {
    text-decoration: underline;
}
  • Descendant Selector: Selects elements that are descendants of a specified element. 
    div p {
    margin-left: 20px;
}
  • Child Selector: Selects elements that are direct children of a specified element. 
    ul > li {
    list-style-type: square;
}
  • Adjacent Sibling Selector: Selects an element that is immediately preceded by a specified element. 
    h2 + p {   
    font-weight: bold;
}
  • General Sibling Selector: Selects all elements that are siblings of a specified element. 
    h2 ~ p {
    color: purple;
}
